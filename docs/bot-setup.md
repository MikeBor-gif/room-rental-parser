# Оформление бота в @BotFather

Разовая настройка «витрины» бота: имя, описание, команды, аватар.
Всё делается в чате с [@BotFather](https://t.me/BotFather).

## 1. Описание (видно до нажатия «Старт»)

Команда `/setdescription` → выбрать бота → отправить:

```
Присылаю новые объявления об аренде комнат и квартир с Kufar, Onliner и Realt — раньше, чем их увидит большинство.

🏙 Минск, Брест, Витебск, Гомель, Гродно, Могилёв
🔎 Фильтры: тип жилья, город, цена
📸 Карточки с фото и прямой ссылкой

Нажмите «Старт» и настройте первый фильтр за 20 секунд.
```

## 2. Короткое описание (в профиле и при пересылке)

Команда `/setabouttext` → выбрать бота → отправить:

```
Новые объявления аренды (Kufar, Onliner, Realt) по вашим фильтрам: комнаты и квартиры, 6 городов, фото и ссылка.
```

## 3. Список команд (меню-подсказка при вводе «/»)

Команда `/setcommands` → выбрать бота → отправить одним сообщением:

```
start - Главное меню
add - Добавить фильтр
filters - Мои фильтры
premium - Премиум-подписка
pause - Приостановить рассылку
resume - Возобновить рассылку
help - Справка
```

(Админ-команды /approve, /reject, /grant, /stats в меню не выносим —
они работают только для ADMIN_CHAT_ID.)

## 4. Аватар

Команда `/setuserpic` → выбрать бота → отправить квадратную картинку
(минимум 512×512). Идея: дом/ключи на однотонном фоне — хорошо читается
в маленьком кружке чата.

## 5. Проверка

1. Откройте бота «глазами пользователя»: описание видно до старта,
   команды подсказываются при вводе «/».
2. `/start` → пройти конструктор фильтра → убедиться, что фильтр появился
   в /filters и в таблице `filters` в Supabase.
3. `/premium` → «Оплатить» → «Я оплатил» → в чат админа пришла заявка
   с кнопками; «Подтвердить» → пользователю пришло «Премиум активирован».
